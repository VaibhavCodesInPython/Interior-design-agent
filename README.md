# Interior-design-agent
An AI-powered basic interior design assistant that generates room designs based on a given floor plan. This project uses an agent-based architecture with structured outputs, guardrails, and tool integrations to create realistic and customizable interior layouts.  
Here, I have built a basic structure for the interior design agent using openai api-key we can also add pricings module for real-time subscriptions.  
## Through this project you can learn about: -  
1.) Transformer-based AI agents  
2.) Function calling & tool usage 
3.) Structured outputs using Pydantic  
4.) Guardrails for output validation  
5.) JSON schema enforcement  
6.) Streamlit UI development
